## 1. dns server
1. make DNS\addon\mysqlConfig.conf  
    ```editorconfig
    [dbConfig]
    host : 127.0.0.1
    user: root
    password: root
    database: database
    charset: utf8
    use_unicode: True
    port: 3306
    ```  
   
2. install mysqlClient [download](https://www.lfd.uci.edu/~gohlke/pythonlibs/#mysqlclient)  
```pip install (download version)```  

3. run server
